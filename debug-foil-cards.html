🔍 Debug - Vérification des cartes Foil

🎯 Étapes de diagnostic

1. Ouvrir la console de développement (F12)

Aller dans l'onglet "Console" pour voir les messages de debug

2. Effectuer une recherche avec "Inclure les Foil" DÉCOCHÉ

Rechercher dans une extension qui contient des cartes Foil

➜ Vous devriez voir : ✨ Filtered out foil cards: X

3. Cocher "✨ Inclure les cartes Foil"

Relancer la même recherche

➜ Vous devriez voir : ✨ Including foil cards in results: Y

➜ Y devrait être plus grand que X

4. Vérifier les résultats affichés

Les cartes avec Foil dans le nom devraient apparaître

Elles doivent être groupées avec leur version normale

🔧 Fonctions de détection Foil

const isCardFoil = (blueprint) => { const cardName = (blueprint.name || '').toLowerCase(); const cardFinish = (blueprint.fixed_properties?.finish || '').toLowerCase(); const cardTreatment = (blueprint.fixed_properties?.treatment || '').toLowerCase(); return cardName.includes('foil') || cardFinish.includes('foil') || cardTreatment.includes('foil') || cardFinish.includes('holo') || cardTreatment.includes('holo'); };

Rechercher "Charizard" dans une extension récente

Rechercher dans une extension Magic

Rechercher "Elsa" ou "Anna"

❌ Si ça ne fonctionne toujours pas

Vérifiez dans la console s'il y a :

  • Des erreurs JavaScript
  • Les messages de debug apparaissent bien
  • Le nombre de cartes change entre coché/décoché
